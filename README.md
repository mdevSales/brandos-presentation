# BrandOS Workshop Presentation

Interactive HTML presentation for the LinkedIn Brand Building Workshop.

## How to Use

1. Open `index.html` in any browser
2. Navigate with:
   - Arrow keys (← →)
   - Spacebar (next slide)
   - Click left/right edges

## Deploy Options

### Option 1: Vercel
```bash
npm i -g vercel
vercel
```

### Option 2: Netlify
Drag and drop this folder to https://app.netlify.com/drop

### Option 3: GitHub Pages
1. Push this repo to GitHub
2. Go to Settings → Pages
3. Set source to main branch

### Option 4: Local
Open `index.html` directly in your browser.

## Presentation Structure

16 slides covering:
- Why LinkedIn branding matters
- Workplace visibility impact
- The BrandOS solution
- 7-step process walkthrough
- Using with AI tools
- Call to action
